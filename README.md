# Anime
 使用Django搭建可视化平台，采集Bilibili上的番剧信息存放在MongoDB，统计各项信息在前端使用Highcharts展示。
 爬虫文件见tools（channel_extracing.py、page_parsing、clean_data.py、download.py）。
